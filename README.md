# Conekta Api Example


Payment example with conekta


# Install


Install global dependencies

```shell
[sudo] npm install sails -g
```

Install local dependencies

```shell
npm install
```

Run

```shell
sails lift
```


# Config

Add your conekta api key in [.sailsrc](./.sailsrc)

```json
  "globals": {
    "conektaApiKey": "my_conekta_api_key_1234"
  }
```
